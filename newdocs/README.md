# Heritage Edition — Documentation Pack

Пакет из четырёх документов, описывающих визуальную идентичность, UI-компоненты, тон голоса и техническую атмосферу Plovxana. Заменяет предыдущие `design_system.md` и `ui_kit.md`.

## Порядок чтения

1. **`visual_identity_heritage.md`** — фундамент. Цвета, слои, физика света, базовая типографика. Читать первым.
2. **`ui_kit_material.md`** — компоненты. Рамки, кнопки, карточки, типографическая шкала. Зависит от (1).
3. **`copy_strategy_nomad.md`** — тексты. Заголовки, микрокопи, банк фраз, запрещённый словарь. Независим от кода, но наполняет компоненты.
4. **`atmosphere_config.md`** — конфиг атмосферы. Карта ассетов, blur/opacity-константы, real-time-состояния. Зависит от (1) и (2).

## Стек

Next.js 14 + Tailwind CSS. Шрифты: Playfair Display (serif), Inter (sans). Анимации: Framer Motion. Изображения: `next/image`.

## Промпт для Cursor (копировать и вставлять)

```
Проект: Plovxana (ресторан центрально-азиатской кухни).
Контекст: все 4 документа пакета Heritage Edition у тебя в этом репо.
Задача: собрать /app/page.tsx как композицию секций Hero → About → Menu → Gallery → Reserve → Footer.
Правила:
1. Используй токены и компоненты строго из ui_kit_material.md. Не изобретай новые.
2. Значения blur/opacity бери из config/atmosphere.ts (см. atmosphere_config.md §2.1).
3. Все тексты — из copy_strategy_nomad.md §2 (Headline Matrix) и §3 (Microcopy).
4. Перед коммитом прогони компоненты по чеклисту из visual_identity_heritage.md §8.
5. Запрещено: эмодзи, rounded-lg на золотых рамках, font-bold, красный #FF0000 в ошибках, спиннеры, text-white, bg-black.
```

## Что ещё понадобится перед стартом разработки

- Профессиональные фото (6–8 шт. для bokeh + галереи) — бриф в `atmosphere_config.md §1.3`
- Финальные SVG-орнаменты (кошкар-муйиз + 6 вариантов тарелок)
- Папка `/public/textures/` с четырьмя текстурами (список в `atmosphere_config.md §1.1`)

Пока этого нет — всё равно можно собирать структуру на заглушках по макету.
